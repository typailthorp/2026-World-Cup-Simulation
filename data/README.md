# Data Sources

This project uses two primary data sources:

- Historical international match data: Kaggle
- https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017?resource=download
- 2026 World Cup fixture data: Kaggle
- https://www.kaggle.com/datasets/areezvisram12/fifa-world-cup-2026-match-data-unofficial
- FIFA rankings data: scraped from publicly available sources  

## Notes

- Raw datasets are not included in this repository due to size and licensing considerations.  
- To run this project, download the datasets and place them in a local `data/` folder.  
- Update file paths in the notebook if needed to match your local setup.
